# Lorentz Arena

*[English version](README.md)*

**特殊相対論がゲームの仕組みになっている**マルチプレイヤー対戦アリーナ。

## Why this exists

特殊相対論は普通、静的な時空図と代数で教えられる。このプロジェクトは、固有時・光円錐・同時性・因果律といった核概念を、「導出する」のではなく「遊んで体感する」ものに変えることを目的にしている。相対論的速度で動き、弾は光的世界線を進み、そして画面に見える他プレイヤーのマーカーは「今そこにいる位置」ではなく「そこから出た光が観測者に届いた位置」になる。避けるには光子が届く前に動き、狙うには光 (そしてレーザー) が到達する時点で相手がどこにいるかを予測する必要がある。

紙のミンコフスキー図を書いていて「この中にいるってどんな感じだろう」と思ったことがあるなら、それを体験するためのもの。

## A concrete moment

他プレイヤーのマーカーが (10, 0) に見える。そこにレーザーを撃つ。レーザーは光的世界線を進むので、相手が予測可能な進路を維持しているときだけ命中する。ゲームは相手側に淡い未来光円錐交点マーカーをレンダーする — これは「レーザーがどこに向かっているか」のヒント。相手は直角方向に小さく加速し、弾は外れる。こちらには過去光円錐の交点三角形 (濃い方) として「実際に到達したレーザー」が後から可視化される。相手の未来光円錐に自分が入ると操作が凍結する：その時点の相手の「現在位置」に基づいて行動したら超光速情報を使うことになってしまうので。

## Core concepts

- **過去光円錐レンダリング** — 各オブジェクトは、観測者の過去光円錐が世界線と交わる位置に描画される。セグメントごとに解析的な二次方程式を解く
- **因果律ガード** — 他プレイヤーの未来光円錐内にいると操作凍結。その位置情報に基づいて動くと光速因果律違反になるため
- **Target-authoritative kill** — 被弾判定はターゲット本人が行う。ホスト権威による「弾が当たったかどうか」の desync を排除
- **静止系切替** — シーン全体にローレンツブーストを掛けて、自分が常に静止して見えるようにする

数式導出は [2+1/README.md](2+1/README.md#relativistic-algorithms)。設計判断の根拠は [2+1/DESIGN.md](2+1/DESIGN.md)。
